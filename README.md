## Description

A chat website example\
Written using Nestjs, Vue 3, socket.io and peerjs

## Installation

```bash
$ npm i
$ npm run seed
$ cd ./client
$ npm i
$ npm run build
```

If you get sqlite3 not found error
```
npm i sqlite3@5.0.2
````

## Running the app (dev mode)

```bash
$ npm run start:dev
```


```
$ cd ./client
$ npm run serve
```
